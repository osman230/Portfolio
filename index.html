Hello, My name is Aisha Osman and this is a portfolio of projects that I've created.

Run Buddy




Run Buddy is a website that I set up for an assignment called Run Buddy. This website is complete with icons, a navigation bar, a map, and a form that can be filled out by the customer in order to sign up.


Run Buddy 2.0




Run Buddy 2.0 is an updated version of the Run Buddy website. Adding onto the features of the previous version of the Run buddy website, this version adds a navigation bar that stays at the top of screen as you scroll as well as a "contact us" section.


Quote Battle




Kanye guess is a game created for a group project. The premise of the game is to guess whether a generator quote is authored by Kanye West or another public figure. This game was created using HTML, CSS framework BULMA, and Javascript. The game includes a timer, and a highscore modal.
